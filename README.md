# Intermediate GitHub Workshop: Practical Publishing with Quarto and GitHub Actions

This repository contains the instructions and slides for the 2026 Intermediate GitHub workshop. Participants will use the GitHub Flow to customize a Quarto website and publish it to GitHub Pages with a GitHub Actions artifact deployment.

### Prerequisites

#### Must Haves
- A Laptop with
  - Git installed
  - A Git client you are familiar with (Git CLI, RStudio, VS Code, GitHub Desktop, etc.)
- GitHub Account
- GitHub 2FA Enabled (required for contributing to repositories)
  - Save some of your recovery codes somewhere secure like a password manager
  - Consider adding a backup 2FA method to prevent lockouts)
- A Basic Understanding of GitHub Flow and the GitHub.com user interface.
  - We won't be covering this material in depth.
  - It is assumed that you either attended or have the equivalent experience of the [Introduction to GitHub Workshop](https://github.com/cascadiarconf-gh-workshops/intro-github-1)

#### Optional
- Familiarity with Quarto and R is helpful but not required
- Familiarity with Git, GitHub Desktop App, and GitHub Actions is helpful but not required
